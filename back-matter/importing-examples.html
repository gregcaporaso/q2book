12. Appendix: Examples illustrating importing data into QIIME 2¶

This document will ultimately pull content from the QIIME 2 documentation’s importing guide. Refer there for now.

12.1. Discovering the available importable semantic types¶

Plugins define semantic types, so the semantic types that can be imported depends on what plugins you have installed. To discover what semantic types can be imported in your QIIME 2 deployment, you can run the following command:

qiime tools import --show-importable-types
